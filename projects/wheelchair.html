Shared Autonomy Wheelchair Navigation

Hierarchical Control Framework with NMPC and Multi-point Virtual APF

OpenNav NMPC Virtual APF ROS/ROS2 C++

The Engineering Challenge

Developing an assistive mobility system requires balancing user agency with autonomous safety. The challenge is to process imperfect user commands and execute smooth, collision-free trajectories in cluttered environments, accounting for the full physical geometry of the wheelchair.

Experimental Validation

Dynamic shared-control: blending user intent with NMPC trajectory optimization and APF safety.

The Navigation Pipeline

We implemented a hierarchical architecture to ensure robust navigation:

  • Autonomous Target Discovery: Using OpenNav and RGB cameras to identify candidate goals when user input is missing or uncertain.
  • NMPC Optimization: A Nonlinear Model Predictive Controller that generates feasible trajectories based on user intent and kinematic constraints.
  • Multi-point Virtual APF: A reactive layer modeling the full wheelchair shape. By handling immediate obstacle repulsion, it significantly reduces the computational burden on the NMPC.

Key Contributions

The integration of predictive and reactive control layers results in:

  • Command Recovery: The system identifies and corrects dangerous or ambiguous user inputs via semantic context.
  • Geometric Awareness: The Multi-point APF strategy (virtual sensors) ensures the entire wheelchair body avoids collisions, not just a single point.
  • Real-time Performance: Efficient C++/ROS implementation

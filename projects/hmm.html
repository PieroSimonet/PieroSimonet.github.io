Context-Aware Intent Decoding

A Generalized Probabilistic Framework for Human-Robot Interaction

HMM Bayesian Inference ROS C++ Multisensory Fusion

The Scientific Challenge

Human intention signals—whether derived from neural activity, eye-tracking, or low-bandwidth interfaces—are inherently noisy and ambiguous. Traditional decoders often fail because they treat the user input in isolation, ignoring the structured environment where the robot operates and where only specific actions are meaningful at any given time.

Experimental Validation

The framework in action: resolving command ambiguity by fusing real-time input with environmental affordances.

Probabilistic Fusion Approach

We implemented a Hidden Markov Model (HMM) architecture that treats user intent as a latent variable, updated through:

  • User Input Evidence: Probabilistic distribution of commands (e.g., BCI, Eye-tracking, Gestures).
  • Environmental Priors: Object affordances and target locations detected via LiDAR and Vision.
  • Task Constraints: Transition probabilities based on the logical flow of the assistive task.

Key Contributions

By closing the loop between the human and the robot's perception, the framework:

  • Disambiguates Noise: Corrects "false positive" commands that don't align with the environment.
  • Reduces Cognitive Load: The robot "anticipates" likely goals, requiring less precise input from the user.
  • Input Agnostic: Demonstrates scalability across different assistive interfaces.

Shared-Control Wheelchair

ROS2 C++ NMPC Python

Overview

This project focuses on the development of a smart wheelchair navigation system that combines user intentions (decoded via BCI or joystick) with autonomous obstacle avoidance using Nonlinear Model Predictive Control (NMPC).

Key Contributions

  • Developed the ROS2 middleware for real-time sensor fusion.
  • Implemented a custom NMPC controller for smooth navigation in cluttered environments.
  • Integrated an intention decoding layer based on EEG signals.

Technologies Used

The system runs on Ubuntu 22.04 with ROS2 Humble. We used Gazebo for initial validation and then deployed the stack on a modified Sunrise medical wheelchair equipped with LiDAR and depth cameras.
